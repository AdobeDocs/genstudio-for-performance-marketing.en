# Adobe GenStudio for Performance Marketing Integrations

GenStudio for Performance Marketing offers integrations with other powerful design products to provide a seamless experience to marketers who rely on those products. Here you'll find information about the integrations that are available and how to use them.

Explore the plugins that GenStudio for Performance Marketing offers, with other Adobe products and beyond, to streamline your design workflows.

## Figma plugin

The GenStudio for Performance Marketing Figma plugin adds a new panel to the Figma application that allows users to perform GenStudio for Performance Marketing actions, like generating on-brand content.

Learn more about the Figma plugin in the [Figma plugin](figma-plugin.md) page.

## Photoshop plugin

The GenStudio for Performance Marketing Photoshop plugin adds a new panel to Adobe Photoshop that allows users to perform GenStudio for Performance Marketing actions, like generating on-brand content.

Learn more about the Photoshop plugin in the [Photoshop plugin](photoshop-plugin.md) page.
