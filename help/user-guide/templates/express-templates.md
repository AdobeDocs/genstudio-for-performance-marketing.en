# Using Adobe Express templates in GenStudio


GenStudio for Performance Marketing can use templates that have been created and designed in Adobe Express. Bring branded assets from Adobe Express and use GenStudio's powerful tools to integrate them in compelling marketing campaigns and Experiences.


> [!IMPORTANT]
>
> This is currently a beta feature. If you're interested in using Adobe Express templates in GenStudio for Performance Marketing, contact your Adobe representative to understand the enrollment options.


This guide explains the requirements and the features with templates from Adobe Express.

## About templates in Express

In Adobe Express, [new documents can be created using existing starter templates](https://helpx.adobe.com/express/web/documents-and-presentations/text-flow-template.html?x-product=Helpx%2F1.0.0&x-product-location=Search%3AForums%3Alink%2F3.7.5) that are provided in the application, or with [custom templates that can include helpful brand restrictions](https://helpx.adobe.com/express/web/brands-libraries-projects/create-manage-brands/edit-shared-template.html) like:

- Locked elements so they can't be edited
- Element style controls for brand alignment

Custom templates can be brought into GenStudio for Performance Marketing and used in campaigns and experiences. Any brand restrictions set in Abdoe Express are included in the template file and will also be applied in GenStudio for Performance Marketing.

Use [the Adobe Express instructions to create a custom template with brand restrictions](https://helpx.adobe.com/express/web/brands-libraries-projects/create-manage-brands/template-control.html).
